este diretório é para armazenar cópias de backup para os dados e os os databases

dumps = cópias de segurança do banco de dados. O DBMS gera SQL para recuperar a estrutura do banco e dos dados.

Exportar dump -> cria e exporta arquivos SQL que definem a estrutura do database (tabelas,colunas,chaves,etc) e dos dados (registros).
Importar dump -> importa arquivos SQL para definir a estrutura do banco e dos dados.

No phpMyAdmin, clicar na casinha antes de exportar um dump. Ao exportar, ele será baixado na pasta downloads.
